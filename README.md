# Asteroids

A clone of the classic arcade game built using Python and Pygame. I built this project to better understand vector math and game loops. 

## Prerequisites
- Python 3.10+
- Pygame-ce

## Setup
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the game: `python3 main.py`

## Controls
- W/S to move forward/backward
- A/D to rotate left/right
- Spacebar to shoot

## Key Features
- Custom collision detection system using circles.
- Vector-based movement 
- Sprite groups for managing game objects
